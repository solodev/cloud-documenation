# Add Deployment

Adding a deployment in Solodev Cloud is a simple process and can be completed in minutes by following the steps below.

<img src="../../../images/adddeployment.jpg" alt="adddeployment" style="width: 100%; display: block"></a>

**Name** | **Description** 
:--- | ---
Deployment Selection | Select the deployment you are interested in by clicking on the button on the product card.

!!!Note:
Some deployments have a purchase option, while others require that you contact Solodev by filling out the contact form. Some are already published in Solodev Cloud and are in the catalog, and you can launch them to your Solodev Cloud. Below is an example of the **Video Platform** deployment that can be launched immediately without purchasing or contacting Solodev.
!!!

<img src="../../../images/adddeployment2.jpg" alt="adddeployment2" style="width: 100% display: block"></a>

**Name** | **Description** 
:--- | ---
Install | Click this button to install the deployment.

!!!Note:
The next steps required to install the deployment depend on its specific characteristics.
!!!
