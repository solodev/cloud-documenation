# Site Overview

Site Overview gives you a comprehensive overview of your site with convenient access to make edits.

At the top, there are three icons, which can be used to change the site view. Maximize the window using the icon of doubled arrows on the right.

**Name** | **Description** 
:--- | ---
<a href="/pages/sites/update/">Update</a> | Edit site settings or delete the site.
<a href="/pages/sites/permissions/">Permissions</a> | Manage your permissions.
